The Informer is a comprehensive open-source solution for real-time extraction and distribution of audio features. The system comprises a VST3/AU/LV2 plugin, a standalone application, a VCV Rack module and externals for Max/MSP, all computing a wide range of time-domain and frequency-domain sound descriptors. An accompanying C++ header-only library (with Python and JavaScript bindings) makes these computational algorithms accessible to third-party developers.

For a detailed overview of the available descriptors, check the official project page on GitHub: [https://github.com/valeriorlandini/theinformer](https://github.com/valeriorlandini/theinformer).

This Max/MSP package features a handful of C++ coded externals that can compute time and frequency domain audio descriptors both on real-time audio streams and on buffer (with any number of channels).
